# Medical Imaging Tumor Classifier

Binary ML classifier that detects malignant vs. benign tumors from lung scan images.

Built using Python, TensorFlow, and scikit-learn as part of Stanford's Inspirit AI program. 
The model is trained on labeled lung scan data and evaluated on classification accuracy 
across both tumor types.

**Stack:** Python, TensorFlow, scikit-learn, Jupyter Notebook  
**Concepts:** Binary classification, image preprocessing, model evaluation
